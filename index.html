Tickets on steroids

Ticketland is the first platform offering programmable tickets powered by blockchain technologies

We are re-imagining tickets

What we're offering is beyond paper tickets - It's a whole new ticketing experience!

Key-less Wallet

Users can simply sign up using web2 type of solutions. No need to download any crypto wallet browser extentions. Use the same credentials to access your wallet both on mobile and web. No private key or mnemonic backups.

Ticket programability

Tickets are not static. They can evolve and change representation based on code added. Each ticket is a fully fledged dApp that can be programmed as any other web app.

Multiple sale types

We support various sale type including free, fixed price, whitelist, refundable and some other exotic ones such as dutch auctions.

Automatic code execution

Custom code that is executed when some conditions are met e.g. refund on flight delays, canceled concert etc.

Fan analytics

Proof of attendance is first class citizen in Ticketland. Third party tools can be built to create communities of members of common interests e.g. Rolling Stones fans.

Secondary market

A combination of p2p and AMM on-chain market that ticket holders can use shall they decide to sell their tickets. The program will make sure that tickets are not re-sold at an over-inflated price.

Buy with fiat            

We have integrated credit and debit card payments which will ease the process of purchasing NFT tickets.

Memorabilia marketplace

Provide on-chain memorabilia to fans that have attended an event e.g. post-event videos, backstage etc.

On-chain affiliate programs

Event organizers can choose partners that will promote the ticket sale of their event. The process will be driven by on-chain programs.

On-chain metadata

Each NFT will have off-chain and on-chain dynamic metadata. e.g. has user attended the given event? Metadata are part of your ticket history.

Tickets are SAFU                

Tickets are non-transferable NFTs which means that even if one steals user's wallet, he would still be unable to prove he is is the original owner of the tickets as wallet are associated with physical identities via Zero-Knowledge cryptography.

Event Crowdfunding

Allow smaller entities to crowdfund the organization of their events. Early supporters will be granted an extra sponsorship NFT which like the ordinary NFT will be part of their ticket history.

Private events            

Ticketland support private events which can be distributed to a close group of attendees defined by the event organizer.

Low service fees            

Utilizing a public blockchain allows Ticketland to offer significantly lower fees as compared to the web2 counterparts.

Fast transaction speed

Ticketland is built on top of the Solana blockchain which has a proven record of high transaction throughput which allows this to be comparable in speed with the web2 counterparts.

Ticketland            

The name of the product is no coincidence. Ticket land offers a SAFU place for your tickets. No one can erase your ticket history. Ticketland is similar to what the photo album is for your daily memories - a memorabilia of the events you attended in the past.
